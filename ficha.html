Ítem Descripción
Nombre del sistema Sistema de Gestión de Inventarios (SGI)
Versión 1.0

Requisitos Funcionales

RF1 - Registrar productos

Permitir registrar nuevos productos al inventario.

RF2 - Editar productos

Modificar la información de productos existentes.

RF3 - Eliminar productos

Eliminar productos del sistema.

RF4 - Registrar pedidos

Permitir ingresar pedidos asociados a productos.

RF5 - Generar reportes

Crear reportes de inventario, movimientos y pedidos.

RF6 - Gestión de usuarios

Administrar roles y accesos para cada usuario.

RF7 - Autenticación

Permitir ingreso con usuario y contraseña.

RF8 - Control de stock

Notificar bajo nivel de inventario.

Requisitos No Funcionales

RNF1 - Seguridad

Proteger la información mediante autenticación y cifrado.

RNF2 - Usabilidad

Interfaz amigable, simple y fácil de usar.

RNF3 - Rendimiento

Respuestas rápidas en operaciones frecuentes.

RNF4 - Escalabilidad

Capaz de crecer en número de usuarios o productos.

RNF5 - Compatibilidad

Funciona en diferentes navegadores y dispositivos.
