Ficha Técnica del SGI

En esta sección se describe la arquitectura técnica del Sistema de Gestión de Inventarios, incluyendo las tecnologías utilizadas y las características clave del sistema.

Características Técnicas

Componente Descripción
Backend Node.js, para el desarrollo del servidor y la lógica de negocio.
Frontend React, para una interfaz de usuario dinámica y moderna.
Base de Datos MySQL, para la gestión de datos relacionales y consultas eficientes.
Arquitectura Modelo Vista-Controlador (MVC) para mantener una estructura modular.
