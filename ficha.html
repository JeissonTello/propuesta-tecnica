Ítem Descripción
Nombre del Sistema Sistema de Gestión de Inventarios (SGI)
Versión 1.0

Requisitos Funcionales

RF01 - Registrar productos

Permitir el registro de productos nuevos con sus respectivas características (nombre, código, categoría, cantidad, proveedor, etc.).

RF02 - Consultar inventario

Mostrar el listado actualizado de productos almacenados en el inventario, permitiendo filtros por categorías.

RF03 - Actualizar información

Permitir la modificación de la información de productos registrados.

RF04 - Eliminar productos

Permitir la eliminación lógica de productos del inventario.

RF05 - Gestión de usuarios

Permitir el registro, autenticación y control de acceso por roles de los usuarios del sistema.

RF06 - Generar reportes

Generar reportes sobre entradas, salidas y estado del inventario en formatos PDF o Excel.

RF07 - Notificaciones

Enviar alertas cuando el stock de un producto llegue a su cantidad mínima.

RF08 - Registrar pedidos

Permitir registrar y hacer seguimiento de pedidos realizados a proveedores.

RF09 - Control de inventario por lote

Permitir identificar productos por lotes para un mejor control de vencimientos o trazabilidad.

RF10 - Auditoría de cambios

Registrar todas las acciones relevantes realizadas por los usuarios en el sistema para fines de auditoría.

Requisitos No Funcionales

RNF01 - Seguridad

El sistema debe garantizar la seguridad de la información mediante autenticación, roles y cifrado de contraseñas.

RNF02 - Rendimiento

El sistema debe cargar en menos de 3 segundos y manejar hasta 1000 transacciones simultáneas.

RNF03 - Usabilidad

La interfaz debe ser intuitiva, responsiva y amigable para usuarios con conocimientos básicos.

RNF04 - Escalabilidad

El sistema debe poder ampliarse para adaptarse a nuevas funcionalidades sin modificar su estructura base.

RNF05 - Disponibilidad

El sistema debe estar disponible el 99.9% del tiempo con tolerancia a fallos y respaldos automáticos.

RNF06 - Mantenibilidad

El sistema debe contar con una arquitectura modular para facilitar el mantenimiento y actualizaciones.

RNF07 - Compatibilidad

El sistema debe funcionar correctamente en los principales navegadores web modernos.

RNF08 - Portabilidad

Debe poder ejecutarse en múltiples plataformas, incluyendo Windows, Linux y macOS.
