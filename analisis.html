Conclusion y Seleccion del Proveedor

Tras el análisis de costos, se determinó que el proveedor más adecuado para este proyecto es Lenovo, con su equipo IdeaPad Slim 3 15IRH8, equipado con:

  • Procesador Intel Core i7
  • 16 GB de RAM
  • Disco duro sólido 1 TB
  • Precio estimado: $4.500.000 COP

En cuanto al software, se optará por herramientas gratuitas y de código abierto siempre que sea posible, priorizando:

  • Licencia Windows 10 Pro
  • IntelliJ IDEA como entorno de desarrollo profesional

Stack Tecnologico Seleccionado

Sistema Operativo

Windows 10 Pro, ideal para aplicaciones empresariales específicas y entornos profesionales.

Base de Datos

MySQL o PostgreSQL, bases de datos robustas, confiables y ampliamente utilizadas.

IDE

Visual Studio Code (Open Source) y IntelliJ IDEA para proyectos de mayor envergadura.

Servidor Apache con opción de migrar a AWS, Azure o Google Cloud según el crecimiento.

Python y Java, ambos open source, conocidos por su versatilidad y soporte en proyectos empresariales.

Framework Backend

Django, para la creación de APIs RESTful y gestión backend robusta en Python.

Framework Frontend

React.js, framework moderno para interfaces web rápidas, dinámicas y reactivas.
