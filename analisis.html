Conclusión con selección del proveedor más adecuado

Tomando en cuenta los resultados del análisis de costos, el proveedor más adecuado para este proyecto es Lenovo, con su portátil IdeaPad Slim 3 15IRH8, que incluye un procesador Intel Core i7, 16 GB de RAM y un disco duro sólido de 1 TB, con un valor aproximado de $4.500.000 COP.

En cuanto al software, se optará por herramientas gratuitas y de código abierto siempre que sea posible. No obstante, se contemplan dos productos de pago esenciales: la licencia de Windows 10 Pro y el entorno de desarrollo IntelliJ IDEA, debido a su compatibilidad y eficiencia para el desarrollo de software.

Stack tecnológico seleccionado

Para el desarrollo del Sistema de Gestión de Inventarios, se seleccionaron las siguientes tecnologías, considerando su robustez, licencias abiertas y facilidad de implementación:

  • Sistema Operativo: Windows 10 Pro.
  • Base de Datos: MySQL o PostgreSQL, ambas muy robustas y de código abierto.
  • IDE: Visual Studio Code (open source) y IntelliJ IDEA (versión paga).
  • Servidor: Apache, y a futuro dependiendo del crecimiento se evaluará AWS, Azure o Google Cloud.
  • Lenguajes de Programación: Python y Java, ambos open source y muy populares.
  • Framework Backend: Django para la gestión de datos y creación de APIs RESTful.
  • Framework Frontend: React.js para el desarrollo de interfaces dinámicas.
  • Librerías de Datos: Pandas (Python) para el manejo y análisis de datos.
