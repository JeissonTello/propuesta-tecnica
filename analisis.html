Análisis del Sistema de Gestión de Inventarios (SGI)

En esta sección se detallan los objetivos, los requerimientos y las necesidades que motivan el desarrollo del sistema de gestión de inventarios.

Objetivos del Proyecto

El principal objetivo de este sistema es optimizar el proceso de control de inventarios, proporcionando una plataforma que facilite la gestión de productos, pedidos, proveedores y reportes de stock.

Requerimientos Funcionales

  • Gestión de productos
  • Control de inventarios en tiempo real
  • Generación de reportes automatizados
  • Gestión de proveedores
