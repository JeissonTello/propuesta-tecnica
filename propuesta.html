Sistema de Gestión de Inventarios (SGI)

Inicio Ficha Contexto Modelo Análisis

Objetivo del Proyecto

Desarrollar un Sistema de Gestión de Inventarios (SGI) que optimice el control de productos, el seguimiento de pedidos y la toma de decisiones empresariales mediante una plataforma tecnológica moderna, segura y eficiente.

Justificación

Actualmente muchas empresas gestionan sus inventarios de forma manual o con herramientas poco eficientes, lo que genera pérdidas, errores de stock y retrasos. Con este sistema buscamos digitalizar y automatizar los procesos claves de inventario, generando mejoras significativas en el rendimiento operativo.

Alcance

  • Registrar, actualizar y eliminar productos del inventario
  • Controlar entradas y salidas de productos
  • Gestionar pedidos y proveedores
  • Generar reportes y estadísticas de movimientos
  • Control de usuarios con diferentes roles

Metodología

El desarrollo del sistema seguirá una metodología ágil basada en Scrum, permitiendo entregas parciales, revisión continua con el cliente y ajustes en el proceso. Se utilizarán herramientas modernas como Node.js, React y bases de datos SQL.

Recursos Necesarios

  • Equipo de desarrollo con conocimientos en tecnologías web
  • Computador con software de desarrollo (VS Code, Git, etc.)
  • Base de datos SQL
  • Servidor de pruebas y producción (puede ser local o nube)
